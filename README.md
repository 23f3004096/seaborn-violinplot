# Customer Support Efficiency Analysis

This repository contains a Seaborn violin plot visualization analyzing customer support response times across different channels.

## Contact
Email: 23f3004096@ds.study.iitm.ac.in

## Files
- `chart.py` - Python script to generate the visualization
- `chart.png` - Generated violin plot (512x512 pixels)

## Visualization
The violin plot shows the distribution of response times (in hours) across three support channels:
- Email Support
- Live Chat
- Phone Support

## How to Run
```bash
python chart.py
```

This will generate `chart.png` with the violin plot visualization.
